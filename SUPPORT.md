# Support

## Getting Help

Kindario is a static HTML, CSS, and JavaScript template repository.

For help with the project, use one of these channels:

- Open a GitHub issue for reproducible bugs or clear feature requests.
- Review the repository documentation before asking setup questions.
- Contact the repository owner on GitHub for private or sensitive concerns.

## Before Opening an Issue

Please try to include:

- The page or file affected
- The browser and viewport used
- Steps to reproduce the issue
- The expected result
- Screenshots when the problem is visual

## Security

Do not use public issues for vulnerability reporting. Follow the process in [SECURITY.md](SECURITY.md).

## Scope Reminder

Kindario is a front-end template only. Support does not include deployment, hosting, payment setup, or custom backend work unless those pieces are added separately by downstream users.
